Login
=====

First some imports::

    >>> from pyogp.lib.base.agent import Agent
    >>> from pyogp.lib.base.network.mockup_client import MockupClient
    >>> from pyogp.lib.base.tests.base import AgentDomain


First we create an agent:
    >>> agent = Agent()

Next we establish the credentials:
    >>> agent.setCredentials('firstname','lastname','secret')

Check the credentials:
    >>> agent.credentials
    PlainPasswordCredential for 'firstname lastname'

Now login the agent to the agent domain:
    >>> netclient = MockupClient(AgentDomain())
    >>> agent.login('http://localhost:12345/auth.cgi', 'http://localhost:12345/region', restclient = netclient)

The result is dummy right now and should contain a long dictionary with region info::
    
    >>> agent.agentdomain.seed_cap
    <SeedCapability for http://127.0.0.1:12345/seed_cap>

    >>> agent.agentdomain.connectedStatus 
    True

    >>> agent.region.details
    {'sim_port': 12345, 'region_seed_capability': '/region_seed_cap', 'sim_ip': '127.0.0.1'}
