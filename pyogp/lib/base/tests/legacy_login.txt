Login
=====

First some imports::

    >>> from pyogp.lib.base.credentials import PlainPasswordCredential
    >>> from pyogp.lib.base.legacy_login import AgentDomain
    >>> from pyogp.lib.base.regiondomain import LegacyLogin

First we create some credentials::
    
    >>> login_params = PlainPasswordCredential('Firstname', 'Lastname', 'secret')
    >>> login_params['start'] = 'uri:Hazzard County&128&128&128'

Then we need some legacy login endpoint to connect to. 
    
    >>> login = LegacyLogin('http://localhost/cgi-bin/login.cgi')

Now we can use both to get an avatar object (this totally needs to change)::
    
    >>> avatar = login.login(login_params)

The result is dummy right now and should contain a long dictionary with region info::    
    >>> avatar.region.details['sim_ip']
    '127.0.0.1'

    >>> avatar.region.details['seed_capability']
    'http://127.0.0.1:12345/seed_cap'

    >>> avatar.region.details['sim_port']
    12345

This needs to be worked on to be a region and not an avatar.
